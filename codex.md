# OTT FOR MBTI 서비스 기획서

## 서비스 개요

**서비스명:** 넷플릭스 취향 MBTI (가칭)  
**컨셉:** 내가 본 콘텐츠 8개에 별점을 매기면, 콘텐츠 취향을 바탕으로 MBTI풍 성향을 추정해주는 서비스  
**목표:** 짧은 평가 경험 → 취향 성향 분석 → 공유 가능한 결과 카드 → OTT/콘텐츠 추천

> 주의: 실제 MBTI 진단이 아니라 엔터테인먼트용 취향 분석 결과로 표현한다. Netflix 공식 서비스처럼 보이지 않도록 “비공식 팬/취향 테스트” 톤을 유지한다.

---

## 기획 변경 요약

기존 기획은 `장르 선택 → 영화 이상형 월드컵 → 유사 영화 추천 → OTT 연결` 구조였다.

수정 방향은 월드컵 경쟁 방식보다 사용자의 실제 시청 경험을 더 직접적으로 반영한다.

```
내가 본 Netflix 콘텐츠 선택 → 8개 콘텐츠 별점 입력 → 취향 벡터 분석 → MBTI풍 결과 도출 → 결과 공유/추천
```

### 왜 바꾸는가

- 월드컵은 “둘 중 하나 선택”이라 재미는 있지만 사용자가 실제로 얼마나 좋아했는지 표현하기 어렵다.
- 별점 방식은 사용자가 이미 본 콘텐츠에 대한 선호도를 더 정교하게 담을 수 있다.
- MBTI형 결과는 공유성이 강해 릴스/숏폼/스토리 확산에 유리하다.
- Netflix 콘텐츠를 진입점으로 삼으면 대중적이고 설명이 쉽다.

---

## MVP 핵심 플로우

```
시작 → 콘텐츠 8개 선택/제시 → 별점 평가 → 분석 로딩 → MBTI풍 결과 → 추천 콘텐츠/공유
```

### MVP 상세

- 사용자가 Netflix에서 봤거나 익숙한 콘텐츠 8개를 평가한다.
- 각 콘텐츠는 1~5점 별점으로 평가한다.
- 평가 대상은 영화와 시리즈를 모두 포함할 수 있다.
- 평가가 끝나면 콘텐츠별 태그를 기반으로 사용자의 취향 점수를 계산한다.
- 결과는 MBTI처럼 4글자 코드와 설명으로 보여준다.
- 결과 카드는 이미지처럼 공유하기 쉬운 형태로 구성한다.
- 결과 하단에는 비슷한 취향의 콘텐츠 3~5개를 추천한다.

## 결과 컨셉

### MBTI풍 축 설계

실제 MBTI 문항을 쓰지 않고, 콘텐츠 취향에 맞춘 4개의 축을 만든다.

| 축 | A 타입 | B 타입 | 의미 |
|---|---|---|---|
| 몰입 방식 | E: 자극형 | I: 감정형 | 빠른 전개/스케일을 선호하는지, 인물 감정과 분위기를 선호하는지 |
| 선택 기준 | N: 세계관형 | S: 현실형 | 판타지·SF·상상력을 선호하는지, 현실감 있는 이야기를 선호하는지 |
| 감상 포인트 | T: 구조형 | F: 공감형 | 플롯·반전·완성도를 보는지, 캐릭터·관계·감정을 보는지 |
| 시청 리듬 | J: 완결형 | P: 탐험형 | 깔끔한 결말을 선호하는지, 낯설고 실험적인 취향도 즐기는지 |

예시 결과:

- **ENFP: 도파민 탐험가**
- **INFJ: 감정선 수집가**
- **ENTJ: 서사 전략가**
- **ISFP: 분위기 감식가**

## 별점 기반 분석 로직

### 콘텐츠 태그 구조

각 콘텐츠는 분석용 태그를 가진다.

```ts
type ContentProfile = {
  id: string;
  title: string;
  type: 'movie' | 'series';
  genres: string[];
  traits: {
    stimulation: number; // 자극/속도감
    emotion: number; // 감정선/공감
    imagination: number; // 세계관/상상력
    realism: number; // 현실성
    structure: number; // 플롯/완성도
    relationship: number; // 관계/캐릭터
    closure: number; // 완결감
    novelty: number; // 새로움/실험성
  };
};
```

### 점수 계산

사용자가 준 별점을 가중치로 사용한다.

```
취향 점수 = 콘텐츠 태그 점수 × 사용자 별점
```

축별로 합산한 뒤 더 높은 쪽을 선택한다.

| 결과 축 | 계산 예시 |
|---|---|
| E/I | stimulation 합계 vs emotion 합계 |
| N/S | imagination 합계 vs realism 합계 |
| T/F | structure 합계 vs relationship 합계 |
| J/P | closure 합계 vs novelty 합계 |

### 신뢰도 표현

점수 차이가 작으면 단정하지 않고 “반반형” 뉘앙스로 표현한다.

- 차이 큼: “당신은 확실한 세계관형 취향”
- 차이 중간: “세계관형에 조금 더 가까운 취향”
- 차이 작음: “현실형과 세계관형을 모두 즐기는 혼합 취향”

## 화면 구성

### 메인 페이지 `/`

- 서비스명과 바로 시작 버튼
- “내가 본 Netflix 콘텐츠 8개로 알아보는 취향 MBTI” 메시지
- 최근 인기 결과 타입 일부 노출

### 평가 페이지 `/test`

- 콘텐츠 카드 8개 제공
- 포스터, 제목, 간단한 장르 표시
- 1~5점 별점 UI
- “안 봤어요” 선택 가능
- 8개 중 최소 6개 이상 평가하면 결과 확인 가능

### 결과 페이지 `/result/[id]`

- MBTI풍 코드와 별명
- 취향 설명
- 내가 높게 평가한 콘텐츠 패턴
- 추천 콘텐츠 3~5개
- 결과 공유 버튼
